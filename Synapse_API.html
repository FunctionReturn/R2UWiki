DebuggerManager

Returns the ROBLOX debugger manager.

Arguments: obj DebuggerManager()
Example: print(DebuggerManager())

setnonreplicatedproperty

Sets a property of a object (userdata) without it being replicated to the server, useful for bypassing anticheats.

Arguments: void setnonreplicatedproperty(Instance object, string property, variant newvalue)

gbmt (Removed)

Gets the game __index, __tostring, and __namecall metamethods before synapse overwrites them.

Arguments: table gbmt()
Example: table.foreach(gbmt(), print)

getgc

Returns everything in the garbage collection in the form of a table.

Arguments: table getgc()
Example: print(getgc())

getgenv

Grabs the global enviorment for Synapse.

Arguments: table getgenv()
Example: print(getgenv())

get_instances

Grabs all instances and returns them in a table.

Arguments: table get_instances()
Example: print(get_instances())

getobjects

Custom getobjects in place of the default ROBLOX getobjects.

Arguments: string getobjects(string Path)
Example: game:GetObjects('rbxassetid://364364477')[1].Source --Gets the source to asset 364364477

getreg

Grabs the Lua registry.

Arguments: table getreg()
Example: print(getreg())

getrenv

Grabs the global Roblox enviorment.

Arguments: table getrenv()
Example: print(getrenv())

getscripts

Returns all scripts in the game.

Arguments: table getscripts()
Aliases: get_scripts
Example: print(getscripts())
