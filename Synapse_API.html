Synapse API

Unoffical documentation for the Synapse API.
Official documentation can be found here.

Intro

This is not and official Synapse API page. This page is created to fill in the missing functions. This page's source can be found here.

For starters, all Synapse functions are no longer in the Synapse table. Most functions are mapped to 'traditional' locations, such as writefile, decompile, and loadfile.

The Raindrop library in Synapse's API has been completely removed in the rewrite the synapse.
However, the old Raindrop documentation can be found here.
The Hooking API is partially covered in this document. Full documentation for that is avaliable here. The bit library is fully covered here.

Note: A few functions are missing due to the lack of knowledge and/or documentation.

DebuggerManager

Returns the ROBLOX debugger manager.

Arguments: obj DebuggerManager()
Example: print(DebuggerManager())

setnonreplicatedproperty

Sets a property of a object (userdata) without it being replicated to the server, useful for bypassing anticheats.

Arguments: void setnonreplicatedproperty(Instance object, string property, variant newvalue)

gbmt (Removed)

Gets the game __index, __tostring, and __namecall metamethods before synapse overwrites them.

Arguments: table gbmt()
Example: table.foreach(gbmt(), print)

getgc

Returns everything in the garbage collection in the form of a table.

Arguments: table getgc()
Example: print(getgc())

getgenv

Grabs the global enviorment for Synapse.

Arguments: table getgenv()
Example: print(getgenv())

get_instances

Grabs all instances and returns them in a table.

Arguments: table get_instances()
Example: print(get_instances())

getobjects

Custom getobjects in place of the default ROBLOX getobjects.

Arguments: string getobjects(string Path)
Example: game:GetObjects('rbxassetid://364364477')[1].Source --Gets the source to asset 364364477

getreg

Grabs the Lua registry.

Arguments: table getreg()
Example: print(getreg())

getrenv

Grabs the global Roblox enviorment.

Arguments: table getrenv()
Example: print(getrenv())

getscripts

Returns all scripts in the game.

Arguments: table getscripts()
Aliases: get_scripts
Example: print(getscripts())

getsenv

Grabs the enviorment for the script passed.

Arguments: table getsenv(variant<LocalScript, ModuleScript> script)
Aliases: getmenv
Example: print(getsenv(game.Players.LocalPlayer.PlayerScripts.LocalScript))

NOTE: Unlike ProtoSmasher, Synapse supports either type of script for getsenv/getmenv. Both functions map to the same thing.
NOTE 2: An empty table will be returned if the given script has finished running.

LoadLibrary

Function made to replace the removal of the ROBLOX library function.

Arguments: RbxLibrary LoadLibrary(string libraryName)
Example: print(LoadLibrary("RbxGui"))

loadstring

Takes a string and returns as a function.

Arguments: Variant<nilfunction, Tuple<nil, string>> loadstring(string String)
Example: loadstring("print('successful operation')")()

islclosure

Returns a bool stating whether a function is a lua closure or not.

Arguments: bool islclosure(function f)
Aliases: is_lclosure
Example: print(islclosure(getfenv))

getinfo

Returns a table of data about the provided function. The first parameter is either a level or a function.
The second argument is a string of what your looking for the options are:

  • f, returns the function active at that level.
  • L, returns the currentline the function is on.
  • u, Number of upvalues the function has.
  • s, Where the function was defined. If the function was defined in a string (through loadstring), source is that string. If the function was defined in a file, source is the file name prefixed with a `@´.
  • n, returns the name of the function (constant) if the function doesn't have a name (annoymas function) it returns a empty string.


    • Arguments: table debug.getinfo(function/level thread, string option)
    Example: local funcdata = debug.getinfo(function1, "s")

    validfgwindow

    Returns a boolean based on if the foreground window is Roblox

    Arguments: boolean validfgwindow(void)
    Example: print(validfgwindow())

    isreadonly

    Returns a bool stating whether a table is a read-only or not.

    Arguments: bool isreadonly(table tbl)
    Example: print(isreadonly(_G))

    is_synapse_function

    Returns a bool stating whether a table is a Synapse function or not.

    Arguments: bool is_synapse_function(function f)
    Aliases: is_protosmasher_closure
    Example: print(is_synapse_closure(getrawmetatable))

    newcclosure

    Creates a CClosure from the Lua function specified.

    Arguments: function newcclosure(function luafunc)
    Example: newcclosure(function(x) print(x) end))("Hello world!") --Prints "Hello World"!

    NOTE: Newcclosure will error if you put an already existing cclosure.

    replaceclosure

    Takes a c closure and replaces it with another c closure.

    Arguments: function replaceclosure(cclosure, cclosure)
    Example: replaceclosure(mycclosure, mynewcclosure)

    NOTE: This works the exactly the same as hookfunction.

    require

    Runs the supplied ModuleScript if it has not been run already, and returns what the ModuleScript returned (in both cases).

    Arguments: Variant require(Variant<ModuleScript, int64> module)
    Example: print(require(52045834)) --Outputs Underscore.lua

    dumpstring

    Returns Roblox formatted bytecode of the provided lua function (string form)

    Arguments: string dumpstring(string source)
    Example: bytecode = dumpstring("local function hi() return 'hi' end")

    unlockmodulescript

    Unlocks a given ModuleScript so it can be required via Synapse.

    Arguments: void unlock_modulescript(Object<ModuleScript> signal)
    Example: UNKNOWN

    checkcaller

    Checks if Synapse is calling the function.

    Arguments: bool Checkcaller()
    Aliases: checkcaller, is_protosmasher_caller
    Example: print(CheckCaller())

    setclipboard

    Reads the contents of the Data variable and copies it to the clipboard.

    Arguments: void setclipboard(string Data)
    Aliases: syn.write_clipboard Example: setclipboard("I'm on the clipboard!")

    Decompile

    Grabs the source code for the script specified.

    Arguments: string Decompile(variant<LocalScript, ModuleScript> Script, bool Bytecode = false)
    Aliases: decompile, Decompiler
    Example: print( Decompile(game.Players.LocalPlayer.PlayerScripts.ChatScript)) --Prints out the source code.


    Decrypt (Removed)

    Decrypts String if the key Password is correct.

    Arguments: string Decrypt(string Password, string String)
    Example: print( Decrypt("haxor123", "REPLACE WITH ENCRYPTED FUNCTION")) --Prints decrypted string.

    DownloadString (Removed)

    Downloads the content from the website provided in the Url argument and returns it back.

    Arguments: string DownloadString(string Url)
    Example: print( DownloadString("https://google.com")) --Prints out the result.

    isrbxactive

    Checks if your focused on the roblox window (Actively clicking) the mousepress/keypress use this function.

    Arguments: boolean isrbxactive(void) Example: print(isrbxactive())


    Encrypt (Removed)

    Encrypts String with Password as the key.

    Arguments: string Encrypt(string Password, string String)
    Example: print( Decrypt("haxor123", "print('haxxed you')")) --Prints encrypted string.

    GetConsole (Removed)

    Creates an instance of the console API.

    Arguments: table GetConsole()
    Example: GetConsole():CreateConsole("Test Console")

    NOTE: See further documentation on the Console API here.
    NOTE 2: This API is to be used with the wrapper only. Full execution is not supported (in full execution the console can only be created).

    GetNilInstances

    Goes through ROBLOX's instance list and returns the instances that have their Parent equal to nil.

    Arguments: table GetNilInstances()
    Aliases: GetNilInstance, getnilinstances, get_nil_instances
    Example: print(#(getnilinstances())) --Prints out the number of nil instances.

    GetRawMetatable

    Gets the metatable even if it is locked.

    Arguments: table GetRawMetatable(table tbl)
    Aliases: getrawmetatable
    Example: GetRawMetatable(game) --Returns the game's metatable.

    HookFunction

    Hooks Backup and returns Replacement instead.

    Arguments: function HookFunction(function Backup, function Replacement)
    Aliases: HookFunc, hookfunction, hook, replaceclosure
    Example: HookFunction(game.HttpGet, newcclosure(function(place, link) print(link) end))

    LoadFile

    Reads the contents of a file from the workspace folder and passes it into loadstring. Errors if the file does not exist.

    Arguments: function LoadFile(string Path)
    Aliases: loadfile
    Example: LoadFile("test.lua")() --Loads the contents of "test.lua" and executes it.

    NOTE: LoadFile does not support the loading of bytecode for security reasons.

    ReadFile

    Reads the contents of a file from the workspace folder and returns it back. Errors if the file does not exist.

    Arguments: string ReadFile(string Path)
    Aliases: readfile
    Example: print( ReadFile("test.txt")) --Prints out the contents of "test.txt".

    SetReadOnly

    The bool sets tbl to be read-only or not.

    Arguments: table SetReadOnly(table tbl, bool)
    Aliases: setreadonly, make_readonly, make_writeable
    Example: SetReadOnly(getrawmetatable(game), false)

    WriteFile

    Writes the contents of the Data argument into the Path in the workspace folder.

    Arguments: void WriteFile(string Path, string Data)
    Aliases:writefile
    Example: WriteFile("test.txt", "hello world") --Writes "hello world" to "test.txt".

    NOTE: This function has security checks to prevent abuse. The Path variable must be alphanumeric. (including - and .)
    The following file extentions also cannot be used with WriteFile: .exe, .msi, .ps1, .reg, .vb, .vbe, .vbs, .scr, .bat, .cmd, .ws, .wsf, .wsh.
    NOTE 2: Some other exploits have a Binary argument at the end of this function. Synapse automatically deals with binary information in WriteFile, so this is not required. If Synapse detects this argument being used, it will ignore it.

    getconstant

    Returns the value of a constant from the selected stack based on stack order.

    Arguments: variant debug.getconstant(int level, int stack)
    Example: print(debug.getconstant(1, 1))

    getconstants

    Returns a table of constants from the selected stack.

    Arguments: table debug.getconstants(int stack)
    Example: table.foreach(debug.getconstants(1), print)

    getlocal

    Arguments: variant debug.getlocal(Variant<int, function> container, string localName)
    Example: print(debug.getlocal(int level, "StringtoPrint"))

    getlocals

    Returns the value of a local from the selected stack based on the name.

    Arguments: variant debug.getlocals(int level)
    Example: print(debug.getlocals(myFunction))

    getregistry

    Returns the registry.

    Arguments: table debug.getregistry()
    Example: print(debug.getregistry())

    getupvalue

    Returns the value of a upvalue from the selected container based on the name.

    Arguments: variant debug.getupvalue(Variant<int, function> container, string upvalName)
    Example: print(debug.getupvalue(myFunction, "StringtoPrint"))

    getupvalues

    Returns a table of all upvalues from the selected container.

    Arguments: variant debug.getupvalues(Variant<int, function> container)
    Example: print(debug.getupvalues(myFunction))

    profilebegin

    Marks the starting point of a custom label with the name profileName in the Microprofiler.

    Arguments: void debug.profilebegin(string profileName)
    Example: debug.profilebegin("CustomLabel")

    profilebegin

    Closes the custom label that was created in this stack's Microprofiler.

    Arguments: void debug.profileend()
    Example: UNKNOWN

    setlocal

    Sets the value of a local from the selected stack based on the name.

    Arguments: variant debug.setlocal(int stackIndex, string localName, Variant value)
    Example: print(debug.setlocal(myFunction, "StringtoPrint", "Changed the string"))

    setupvalue

    Sets the value of a local from the selected stack based on the name. Returns a boolean depending on if it succeeded or not.

    Arguments: variant debug.setupvalue(Variant<int, function> container, string upvalName, Variant value)
    Example: print(debug.setupvalue(myFunction, "StringtoPrint", "Changed the string"))

    setconstant

    Sets the value of a constant from the selected stack based on the stack level.

    Arguments: variant debug.setlocal(int stackIndex, int stackIndex, Variant value)
    Example: debug.setconstant(2, 1, "NewFunctionName")

    traceback

    Arguments: variant debug.traceback(Variant<int, function> container, int level)
    Example: print(debug.traceback(myFunction, 1))
